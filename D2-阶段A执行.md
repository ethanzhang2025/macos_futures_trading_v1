## 1. Stage A 目标

- **M6 上线**：Pro 订阅功能 Day 1 收费（生死节点）
- **M9 达成**：Pro 付费 ≥ 500 / 月流水 ≥ ¥1.5-2 万 / 团队续命进入 Stage B

**Persona 聚焦（承自 D1）**：
- **P1** Mac 主观 Pro 交易者（65%）
- **P2** Mac 半职业波段交易者（25%）
- **P3** 文华迁移者（10%，M7-M9 麦语言基础版释放时启动）

## 2. Stage A 产品 MVP 范围

### 设计原则（承自 D1）
**统一按 Pro 实现所有功能 → 不做 Free/Pro 代码分层**。M5 末基于真实内测数据开"上线决策会"，决定哪些功能锁给 Pro、哪些开放给 Free。

**轻量 feature flag 层**（工程量约 1 天）：
- 用本地 `UserDefaults` + 远程 JSON 配置
- 开发期所有 flag = true
- M5 末决策后配置生产端开关

### M1-M6 上线版功能

**图表引擎 · Metal 自研**
- 全市场合约实时行情（CTP Level 1 免费数据）
- 60fps K 线渲染（目标 10 万根 K 线无卡顿）
- 多周期切换（1m/5m/15m/30m/1h/4h/日/周/月）
- 多窗口布局（最多 6 个同屏）

**指标库 v1（56 个）**

| 类别 | 数量 | 代表 |
|------|------|------|
| 趋势类 | 10 | MA / EMA / WMA / DEMA / TEMA / HMA / VWAP / SAR / Supertrend / ADX |
| 震荡类 | 12 | RSI / MACD / KDJ / Stochastic / CCI / W%R / ROC / TRIX / BIAS / PSY / DMI / CMO |
| 量价类 | 8 | Volume / OBV / MFI / CMF / VR / PVT / ADL / VOSC |
| 波动率/通道类 | 8 | BOLL / ATR / KC / Donchian / StdDev / HV / Price Channel / Envelopes |
| 支撑阻力/结构类 | 6 | Pivot Points / ZigZag / Ichimoku / Andrew's Pitchfork / Fractal / Elliott Wave |
| **期货特有（差异化）** | **12** | **持仓量 / ΔOI / 主力合约切换 / 涨跌停板线 / 夜盘日盘分界 / 交割日倒计时 / 结算价线 / 多空力量 / 成交密度 / 价差线 / 基差线 / 强弱对比** |

**画线工具 v1（6 种）**
- 趋势线 / 水平线 / 矩形 / 平行通道 / 斐波那契回调 / 文字标注

**自选管理**
- 多分组（无上限）
- 每组合约无上限
- 拖拽排序

**复盘分析 v1（8 张图）**

| # | 图表 | 说明 |
|---|------|------|
| 1 | 月度盈亏走势 | 累积曲线 |
| 2 | 盈亏分布直方图 | 按金额/频次 |
| 3 | 胜率曲线 | 累计胜率随时间 |
| 4 | 品种盈亏矩阵 | 热力图 |
| 5 | 持仓时间分布 | 多久开仓多久平仓 |
| 6 | 最大回撤曲线 | 风控核心 |
| 7 | 盈亏比分析 | 平均盈利 ÷ 平均亏损 |
| 8 | 时段分析 | 早盘/午盘/夜盘 |

**数据导入**
- 券商交割单 CSV 导入
- 文华交割单格式适配

**CloudKit 架构预埋**
- 数据结构为多端同步预留
- Stage A M7 启用 UI 层同步

**条件预警中心**（新增 · 专业刚需）
- 价格预警（突破 / 跌破 / 到达特定价位）
- 画线预警（趋势线突破 / 斐波那契位触及）
- 波动率/成交量异常提醒（放量、急涨急跌）
- 预警统一面板（状态、触发历史、启用/暂停）
- 通知渠道：App 内弹窗 / macOS 通知中心 / 声音 / （Stage B 加推送到手机）
- 历史预警记录（可回看复盘）

**交易日志**（新增 · 高粘性低门槛）
- 半自动：每笔交易从交割单导入后自动生成日志条目
- 手动补充：原因、情绪状态、计划偏差、学到的教训
- 标签体系 + 搜索
- 月度/季度总结自动生成
- **Stage A 最强粘性功能**（日积月累无法迁移）

**模拟训练**（新增 · 无实盘接入时的自然过渡）
- 接 CTP SimNow 仿真环境（零成本接入）
- 模拟盘练习（独立虚拟账户）
- 训练模式：指定历史场景 + 操作评分
- 纪律检查（计划 vs 实际偏差）

**K 线回放**（新增 · 沉浸式复盘）
- 选历史日期 + 品种，回放任意周期
- 回放控制：播放 / 暂停 / 加速（2x-8x）/ 倒退 / 单步
- 当日成交点叠加标注
- 替代纯报表复盘的"回到当时那一刻"体验

**工作区模板**（新增 · Pro 用户工作流粘性）
- 多窗口布局保存（每套模板记录所有图表/合约/指标/画线）
- 多套模板快速切换（如"盘前分析 / 盘中盯盘 / 盘后复盘"三套）
- 快捷键一键切换
- CloudKit 云同步

### M7-M9 加入（非上线阻塞项）

- **iPad 版**（基础同步看盘，不做完整专业工作流）
- **CloudKit 多端自选同步**
- **麦语言解析器基础版**
  - 自研词法/语法分析器 + AST 执行器（全 Swift 实现）
  - 覆盖常用 30-50 个文华内置函数（MA/EMA/MACD/KDJ/BOLL/RSI/CCI 等主流指标所需）
  - 用户可运行已有自定义指标公式
- **画线扩展至 8 种**（+ 垂直线 / 价格标签）

### Stage A 绝对不做

- ❌ CTP 下单（Stage B）
- ❌ Python 集成 / DataFrame 导出 / ⌘K 命令面板（Stage B）
- ❌ iPad 外接屏专业模式 / Apple Pencil 画线（Stage B）
- ❌ Windows 版
- ❌ iPhone 版
- ❌ 麦语言**完整兼容**与**策略执行引擎**（Stage B 核心战役）

## 4. 技术栈详细选型

### 客户端层

| 层 | 选型 | 原因 |
|----|------|------|
| 语言 | **Swift 6**（严格并发） | Mac/iPad 原生唯一选择 |
| UI 框架 | **SwiftUI 主 + AppKit 精细控制** | SwiftUI 快速构建 + AppKit 处理图表交互复杂场景 |
| **图表引擎** | **Metal 自研** | 核心差异化；不用 SwiftUI Charts（性能不够）/ WebView JS 图表（不原生） |
| 指标引擎 | Swift 原生实现 56 个指标 | 同时作为麦语言内置函数的底层调用 |
| 本地存储 | **SQLite** | 稳定成熟 |
| 云同步 | **CloudKit**（非敏感）+ **阿里云自建**（敏感） | CloudKit 存 UI 布局 / 自选；交易日志等敏感数据走境内阿里云自建。详 StageA补遗 G1 |
| **麦语言解析器**（M7+） | Swift 自研词法/语法/AST | 全 Swift 栈内一致 |

### 后端层

| 层 | 选型 | 原因 |
|----|------|------|
| 语言 | **Go** | 高并发、编译快、部署简单 |
| 用户/订阅服务 | Stage A 极简（几百行 Go） | 只需用户注册/订阅状态/feature flag |
| 数据库 | **PostgreSQL**（生产）或 SQLite（Stage A 初期可用） | 用户和订阅数据 |
| 行情中继 | Stage A 客户端直连 CTP，**Stage B 再加中继层** | 先简化，后优化 |
| 部署 | **阿里云国内单节点**起步 | 合规必需境内 |
| 监控 | **Sentry** + 自建审计日志 | Sentry 对 Apple 平台支持最好 |

### 开发工具链

| 工具 | 选型 |
|------|------|
| AI 编码 | **Cursor + Claude Code** 双用 |
| IDE | **Xcode + Cursor** 双开 |
| 版本控制 | **GitHub 私有仓** |
| CI/CD | **Xcode Cloud** |
| 任务管理 | **Linear** |
| 文档协作 | **飞书文档**（国内流畅）|

## 5. GTM / 冷启动策略

### 主战场 · 合伙人 Hunter 1v1

- **每天 10-20 个目标用户私信**（雪球、知识星球、期货 QQ/微信群、小红书私信）
- **每周 ≥ 30 次深度对话**
- **6 个月累计 1500+ 触达**

### 副战场 · 内容与 ASO

- **少数派投稿**：Stage A 关键节点发 1-2 篇深度评测（M3 末介绍内测 / M6 末介绍上线）
- **即刻 + V2EX**：你偶尔发产品更新帖（1 篇/月，非 IP 化）
- **Mac App Store ASO**：合伙人次要任务，优化关键词（期货、交易、行情、复盘、Mac）

### 放弃的渠道（两人团队做不了）

- ❌ 小红书日常运营
- ❌ 雪球/知乎日更内容
- ❌ B 站 KOL 付费合作（Stage A 没预算）
- ❌ 创始人 IP 矩阵

### 用户邀请闭环

- TestFlight 邀请制：从合伙人 1v1 对话筛选进入
- 付费 Pro 用户加入 VIP 微信群（邀请制，合伙人运营）
- 老用户推荐奖励：推荐 1 人付费 → 推荐人订阅免费延期 3 个月

## 8. Go/No-Go 里程碑

**M3 checkpoint**（详见 D3 月度 standup）
- TestFlight 累计 ≥ 100 + 访谈付费意愿 ≥ 40% → 继续
- 低于红线连续 2 月 → D3 Plan B 会议

**M6 生死节点**
- Pro 订阅上线 + 首个付费用户入账 → Stage A 上半段成功
- 未上线或零付费 → 核心假设破产，必须全面重评估

**M9 Stage B 启动门槛**
- Pro ≥ 500 / 月流水 ≥ ¥1.5 万 → Stage B 启动
- 不达标 → 延长 Stage A 或 Plan B
