## 2. Stage B 产品扩展范围

### 2.1 产品新增（12+ 模块）

**① CTP 下单 · 订单体系**（Stage B 核心战役）
- **基础订单**（M12-M14 必做）：市价 / 限价 / 止损 / 止盈 / 快捷键下单
- **高级订单**（M15-M18）：追踪止损 / OCO / 括号单 / 冰山单
- **算法订单**（M20-M24，视客户需求）：TWAP / VWAP / 阶梯单 / 狙击单
- 技术：Obj-C++ → Swift Package 封装 CTP API

**② 麦语言完整兼容 · 策略执行引擎**（战略武器）
- 函数覆盖：Stage A 末 30-50 → M18 85 → M24 115（95%+ 主流）
- 策略执行引擎：信号生成 → 回测 → 实盘触发
- 话术：Stage A 晚 "你熟悉的交易语言在 Mac 跑起来" → M18 "零成本迁移" → M24 "你的策略，原样运行"

**③ iPad 专业工作流 · 6 大独家场景**
- 出行盯盘 / 会议摆台 / Apple Pencil 画线 / Magic Keyboard 下单 / 外接 4K / Sidecar 协同
- 快期/文华因不在 Apple 生态几乎无法复制

**④ iPhone 伴侣版**
- 基础推送 + 快速看盘 + Widget + Shortcuts
- 不做完整专业工作流

**⑤ 图表扩展**
- 指标库 +100（80 → 180+）
- 画线 +2（垂直线 / 价格标签）+ Apple Pencil 自由画线
- Level 2 行情（可选）
- 多屏专业布局

**⑥ 工作流深化**
- K 线回放：多合约联动
- 条件预警：+ 推送到手机 + Apple Watch
- 交易日志：AI 辅助分析（情绪/偏差识别）
- 模拟训练：历史场景回放训练
- 工作区模板：团队共享（Pro Max）

**⑦ Apple 生态深度**
- Focus 模式（盘中自动屏蔽其他通知）
- Continuity（Mac 与 iPad 接力）
- 预埋 Apple Watch（Stage C）

**⑧ Python / ⌘K 命令面板**（P4 技术型交易者）
- Python 脚本集成（读数据 + 计算指标）
- DataFrame 导出
- ⌘K 全键盘命令面板

**⑨ 文华数据深度迁移**
- 画线数据导入（Stage A 未做）
- 完整指标库兼容（配合麦语言 95%）

**⑩ 行情中继层**
- Stage A 客户端直连 CTP；Stage B 加中继层降低连接成本 + 增强监控

**⑪ B2B2C 多账户视图**（Stage B 末期，视需求）
- 白标定制 / 多账户聚合查看 / 营业部风控端

**⑫ 用户数据导出 + 订阅降级**（承接 Stage A G13/G14）

### 2.2 Stage B 不做（DNA 红线延续）

- ❌ Windows 版（Stage C 视数据再议）
- ❌ 自营 / 沉淀客户资金

## 3. 定价扩展

### 3.1 三层定价正式生效

| 层级 | 价格 | 启用时点 | 含什么 |
|-----|------|--------|-------|
| Free | ¥0 | Stage A M5 末决策 | 基础功能 |
| **Pro** | ¥399/年 或 ¥39/月 | Stage A M6 | 全部 Stage A 功能 + Stage B 新增（除 Pro Max 专属） |
| **Pro Max** | **¥999/年** | **Stage B M12** | + CTP 下单 + iPad 专业工作流 + 麦语言完整 + 多端深度同步 + AI 复盘 + 团队模板 |

### 3.2 升级机制

**Pro → Pro Max**：按剩余天数退补差价（用 Apple IAP 订阅级别升级机制）
- 例：Pro 用户付费后 6 个月升 Pro Max，退剩余 Pro 半年价值 ¥200，补差价 ¥500 - ¥200 = ¥300

### 3.3 B2B2C 代付定价

| 买单方类型 | 单价（按人按月） | 合同最小规模 |
|----------|--------------|------------|
| 期货公司总部 | ¥30-40/人月 | 500 人起 |
| 营业部（优先谈）| ¥35-50/人月 | 100 人起 |
| 大客户团队 | ¥60-80/人月 | 20 人起 |
| 高净值客户服务体系 | ¥40-60/人月 | 100 人起 |

产品不分化：所有 B2B2C 终端用户看到标准 Pro 产品；Stage B 末期视需求加白标。
