### 5.6 WP-44b · UnifiedDataSource 多 period 订阅修复（commit e85ac4a）

#### 问题（之前 demo 暴露）

- `start(instrumentID, period)` 每次都调 `realtime.subscribe(instrumentID, handler)`
- `MarketDataProvider.subscribe` 是 `dictionary[id] = handler` 字典覆盖语义
- 同 instrumentID 多 period 同时订阅时，仅最后一个 period 收到 tick

#### 修复（手术式 · 30 行）

- `start`：仅当此 instrumentID 是首次订阅时调 `realtime.subscribe`（用 `subscriptions.keys` 内省判断）
- `handleTick`：移除 key 参数，按 `tick.instrumentID` 找出所有 matchingKeys，对每个 key 各 dispatch 一次
- `cleanup`：行为不变（仍只在 instrumentID 完全无订阅时 `realtime.unsubscribe`）

#### 向后兼容 + 新增测试（3 个）

- 现有 524 测试 0 修改 0 失败
- multiPeriodSubscribesOnce · 同 RB 多 3 周期 → realtime.subscriberCount = 1
- multiPeriodAllReceiveTicks · 跨 6 分钟 ticks → minute1 ≥5 根 + minute5 ≥1 根
- stopOnePeriodKeepsOthers · 关 minute1 不取消 realtime · 全关后才 unsubscribe

## 11 · 可能踩的坑（v5.0 增量）

> v4.0 已记录的 11 个坑保留有效，本节仅列 v5.0 新增。

| # | 现象 | 原因 | 解决 |
|---|------|------|------|
| 12 | **Linux Sina API 0 tick 返回**（demo / 集成）| Sina 报价端点返回 GBK 编码 + Linux Swift `String.Encoding.ascii` 对 >127 字节返回 nil | `SinaMarketData.swift` line 32 · `.ascii` fallback → `.isoLatin1` fallback |
| 13 | **Swift 6 nonisolated deinit 不能访问 actor 状态** | 严格并发限制 | actor 必须显式 close()；进程退出时 SQLite 文件句柄由 OS 回收 |
| 14 | **同 instrumentID 多 period 订阅仅最后一个收到 tick** | `MarketDataProvider.subscribe` 字典覆盖语义 | UnifiedDataSource WP-44b 修复：仅首次订阅注册 handler + handleTick dispatch 给所有匹配 keys |
| 15 | **BOLL 输出顺序非直觉** | BOLL.calculate 返回 `[MID, UPPER, LOWER]`（非 [U,M,L]）| Demo / caller 注意顺序；标签按数值范围反推 |
| 16 | **SQLite Decimal 精度损失** | SQLite 无原生 Decimal 类型 | Decimal 用 TEXT 存（NSDecimalNumber.stringValue），读时 Decimal(string:) 解析 |
| 17 | **InMemoryStoreTests struct 重名** | Wp23 + Analytics 两边都用 InMemoryStoreTests | 加前缀（InMemoryAnalyticsStoreTests）|
| 18 | **`SQLITE_TRANSIENT` 在 Swift 不可直接用** | C 头里是 `((sqlite3_destructor_type)-1)`，Swift 不能直接表达 | 手动重新声明：`unsafeBitCast(OpaquePointer(bitPattern: -1), to: sqlite3_destructor_type.self)` |
| 19 | **systemLibrary CSQLite 不能含 Swift 文件** | Swift Package 限制 | Sources/CSQLite/ 只放 module.modulemap + shim.h |
