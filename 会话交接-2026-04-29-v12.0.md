## 12 · 可能踩的坑（v12.0 新增 15 个 · v11.0 之前坑保留）

### v11.0 → v12.0 踩坑

1. **AlertWindow Alert 命名歧义**（v12.0 修）：SwiftUI.Alert（deprecated 但仍存在）vs AlertCore.Alert · Mac 编译歧义 · Linux 跳 SwiftUI 块未暴露 · 修：`typealias Alert = AlertCore.Alert`（必须 internal · 不能 private）
2. **AlertHistoryEntry 参数顺序**（v12.0 修）：协议是 `id / alertID / alertName / instrumentID / conditionSnapshot / triggeredAt / triggerPrice / message` · AlertWindow 之前用了错的顺序（triggerPrice/triggeredAt/conditionSnapshot 互换）· Linux 跳 SwiftUI 未暴露 · 修：3 处统一顺序
3. **ChartScene compactMap 推断歧义**（v12.0 修）：`compactMap { hist in guard ... else { return nil } ... }` · Swift 6 严格推断需显式 `-> KLine?`
4. **try? await 未用返回值 warning**（v12.0 修）：service.record 返 Int64? · 不接收会 warning · 加 `_ =`
5. **JournalWindow Decimal LocalizedStringKey deprecated**（v12.0 修）：SwiftUI Text("\(decimal)") 默认走 LocalizedStringKey · macOS 14+ deprecated · 改 `decimal.description` 显式 String
6. **KLine.openInterest 类型**（v12.0 修）：模型是 Decimal · Sina API 返 Int · 转换需 `Decimal(int)`
7. **.gitignore 整目录 .claude/**（v12.0 修）：之前只忽略 settings.local.json · scheduled_tasks.lock 误入仓库 · 修：整目录 + git rm --cached
8. **ReviewWindow 8 图布局裁切**（v12.0 修）：固定 4 列 .flexible(min: 280) · 4×280+3×16=1168 > minWidth 1024 · 默认窗口右侧裁切 · 修：.adaptive(minimum: 260) + minHeight 替代 height
9. **Mac 端 swift run 命令名**（v12.0 发现）：executable 是 `MainApp`（Package.swift line 105）· 不是 `FuturesTerminalApp`（后者只是 struct 名）· 命令：`swift run MainApp`
10. **回放未启动主图大绿块**（v12.0 修）：MetalKLineRenderer.makeViewMatrix x 右边界用 `Float(visible)` 而非 `Float(viewport.visibleCount)` · 单根 bar 时 visible=1 · xRange=[0,1] · bar 实体 0.7 占 70% 屏宽
11. **默认拖拽无效**（v12.0 修）：`bars.count == visibleCount == 200` · pan 范围 = 0 · 修：visibleCount 200→120 + cacheMaxBars 200→500
12. **空数组 vs nil 持久化语义**（v12.0 一致化）：alerts 空数组合法（用户清空意图）· nil 才 fallback Mock · 与 history 加载之前 isEmpty guard 不一致（v12.0 P1-4 修）
13. **K 线 append 多 Task 并发**（v12.0 P0-2 修）：每根 completedBar 一个 Task · SQLite actor 串行写但 Task 提交顺序不保证 · maxBars 截断时可能丢中间根 · 修：klineSaveTask 链式 await
14. **AlertHistory testTrigger 不写库**（v12.0 P0-1 修）：appendHistoryEntry 仅 historyEntries.insert 不调 store.append · evaluator 接入前是唯一 history 信号源 · 重启丢失 · 修：append 调 store
15. **trades 空数组被静默忽视**（v12.0 P0-3 修）：onChange(of: trades) `!newValue.isEmpty` guard 阻止"用户清空"持久化 · 修：去 guard
