### 6.4 SwingPointsDetector 严格大于（batch82）

```swift
isLocalHigh = bars[priorRange].allSatisfy { bar.high > $0.high }
              && bars[afterRange].allSatisfy { bar.high > $0.high }
```

严格大于（>）而非 >= 防平顶/平底产生过多噪声 swing。lookback ≤ 0 / bars 数 ≤ 2*lookback → 空数组防御。
