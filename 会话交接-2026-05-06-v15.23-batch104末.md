## 11 · 可能踩的坑（v15.23 batch70-104 增量）

### batch70-104 新坑

1. **多 cell 同实例 pipeline**（batch70）：每 cell new MarketDataPipeline → 6 个独立 SinaMarketData/PollingDriver/Provider/UDS · 资源浪费但功能 OK · UDS 内部 WP-44b 同 instrumentID 共享 realtime · 网络层去重生效
2. **task 取消时 leak 风险**（batch70）：.task(id:) 取消时 stream 不 emit 则 for-await 不退出 · 老 pipeline 持续 polling · 内存 leak（功能不崩 · 待 batch101+ 优化未做）
3. **Codable 老 cellsJSON 兼容**（batch78/86/91/93/94/97/98/99）：每加新字段必须 decodeIfPresent ?? defaultValue · 漏会导致老用户配置全失（trader 数据丢）
4. **mock data OI 全 0 历史**（batch87）：用户老 cellsJSON cache 里 OI=0 · 切到 OI 副图显示"OI 数据未提供"灰提示线 · 真行情接入后 cache append 才有真 OI
5. **状态字段加太多 toolbar 拥挤**（batch88 重构 Menu 收纳前）：6 个 toggle button + 文字 + 数值 + emoji 在小 cell 折行 · 必须用 Menu 收纳
6. **subChartIcon 系统图标依赖**（batch79/87/95）：person.3.fill / tornado / waveform.path.ecg 需 SF Symbols · macOS 13+ 全部支持 · 老 macOS 不可
7. **ImageRenderer macOS 13+**（batch100）：截图功能依赖 macOS 13+ ImageRenderer API · platforms .macOS(.v13) 满足
8. **font label 颜色 hack**（batch97）：`color.opacity(2)` 让 label 文字透明度高于线条 · 是绕过 SwiftUI Color API 限制的 trick · 实际效果是 alpha 不变（≤1.0 钳制）但视觉对比度通过 weight 调整
9. **BOLL 末根算 race**（batch92）：breakoutBorderColor 内重新算 BOLL 末根（period=20 window）· 与 drawBoll 逻辑独立 · 性能可接受（O(20) 一次）
10. **6 cell 评级 host 重复算**（batch104）：cellsBullSummaryView 遍历 cells × bullScoreAt · 每秒 host body 重渲时算 6 次 · 6 cell × O(120) ≈ 720 ops/s · 可接受
