## 5 · 关键设计沉淀

### 5.1 期权 Phase 6 战略意义

**OptionStrategy 数据模型扩展**：
- 加 `underlyingPositionSize: Int`（默认 0 · 向后兼容）
- 加 `underlyingEntryPrice: Double`（默认 0）
- `payoffAtExpiration` 加标的 MTM PnL（单股语义 · 与 leg 量纲对齐）
- 解锁备兑 / 护跌 / 现货+期权组合策略 · 不只局限纯期权策略

**期权回测 = 静态持有 MTM**：
- 输入 [date, spot, IV] 时序 · 每点用 BS 重定价所有 leg + underlying MTM
- 输出 PnL 曲线 + endingPnL + maxDD + 年化 Sharpe + winRate + best/worstDay
- T → 0 BS 退化为内在值 · 与 leg.payoffAtExpiration 自然对齐
- v1 mock 5 模式：random walk / 上涨 / 下跌 / 横盘 / V 反转

### 5.2 套利 V2 核心：滚动 Z + 状态机

**滚动 Z 计算**：
- 每个点用过去 window 根（含自身）的 mean / σ
- 默认 window=60（≈ 1h@1m / 1d@15m · UI 可调 5-200）
- 不足 window 时返 0（无意义点 · 不显示）

**信号状态机**：
- 无仓位 + Z >= +entryThreshold → short entry
- 无仓位 + Z <= -entryThreshold → long entry
- 持空 + Z <= +exitThreshold → exit
- 持多 + Z >= -exitThreshold → exit
- 持仓 >= maxHoldingBars → 强平（防永久套牢）
- 末尾仍持仓 → 用末样本强平（避免回测 trade 不闭环）

**回测 PnL**：
- 做多价差：pnl = exitValue - entryValue
- 做空价差：pnl = entryValue - exitValue
- 不含交易成本（v2 简化 · v3 加 commission）

### 5.3 行情 V2 多空偏向 bullBias

```
bullBias = (gainers - losers) / total
范围 [-1, +1] · 红多 / 绿空 / 0 平衡
统计 HUD 用进度条可视化（trader 一眼定位市场情绪）
```

## 10 · 可能踩的坑（本 session 新增）

### 10.1 期权 Phase 6.1 铁鹰
1. **strike 必须严格 K1<K2<K3<K4**（违反返 nil · UI 已加自动 remap 防误）
2. **铁鹰是卖方策略**：净权利金为负（净收入）· UI HUD 颜色判断已处理
3. **OptionWindow 4 strike 字段**：用 strategyExtraStrike 作为 K4（其他策略不用）

### 10.2 期权 Phase 6.2 备兑/护跌
1. **underlyingPositionSize = quantity**（不乘 contractMultiplier · 单股语义对齐 leg PnL）
2. **裸卖 call 时 underlyingPositionSize=0**（OptionStrategy init 默认值 · 向后兼容）
3. **payoffAtExpiration 公式**：optionMTM + underlyingMTM = total（不要单独算 leg）

### 10.3 期权 Phase 6.3 回测
1. **T → 0 时 BS 退化为内在值**（max(T, 1e-6) 防除零 · 末样本与 leg.payoffAtExpiration 误差 < 0.5）
2. **Sharpe 用绝对 PnL 差分**（非收益率 · 不依赖初始本金 · PnL 维度比较）
3. **maxDrawdown 是绝对值**（不带负号 · UI 显示要保持正数）
4. **OptionWindow currentStrategy 是 computed**（sheet 拿不到 binding · 用 snapshot 传值）

### 10.4 套利 V2
1. **rollingZScores 不足 window 返 0**（不是 nil · 调用方判断 i >= window-1 才有意义）
2. **状态机末尾若仍持仓 · 末样本强平**（避免回测 trade 不闭环）
3. **做多 vs 做空 PnL 公式相反**：long = exit-entry · short = entry-exit
4. **histogram bin 边界**：lower 含 / upper 不含 · 末桶含（防 max value 落到 binCount 越界）

### 10.5 行情 V2
1. **filter preset 持久化键**：viewState.v1.watchlist.filterPresetRaw（重启保留）
2. **groupSearchText 是 @State**（不持久化 · 切分组保留 · 跨会话清空 · 故意行为）
3. **bullBias 公式**：(gainers - losers) / total · 范围 [-1, +1]
4. **隐藏字段排序触发器**：用 .hidden() + width=0 · 仅供 sortFieldMenu 调用

### 10.6 ChartHitTester
1. **viewport.startIndex 可能为负**（异常输入 · 用 max(0, startIndex) 防御）
2. **barCount=0 / width=0 返 nil**（防除零 · 调用方需 unwrap）
3. **price 一字板（max==min）防除零**：返 height/2（中点 · 不 crash）
4. **viewport vs full 模式**：选错会导致 hit-test 偏 · 主图必须用 viewport 版本

### 10.7 WP-40 P1 session-aware
1. **detectGaps daily 周期不检测**（K 周期本身已大 · 周末缺口在数据层自然吸收）
2. **session vs day 阈值**：session > 2×period · day > 6h（绝对秒数 · 不依赖 period）
3. **isInNightSession 跨午夜**：21:00→02:30 用 endM<startM 处理
4. **KLineAxisView 标签智能避让**：raw idx 落在 gap ±1 → 平移到 gap 后侧（显示新段开盘时间）
