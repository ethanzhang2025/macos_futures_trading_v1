## 5 · 关键设计沉淀（本 session 新增）

### 5.1 ComboAnomalyAggregator 算法（v15.70）

```
按 instrumentID 分组 · ≥ minKinds 类（默认 3）保留
severity 数量加权：avg × (1 + 0.2 × max(0, kindCount - 3))，clamp [0, 100]
- 3 类 → ×1.0 (60 分 → 60)
- 4 类 → ×1.2 (60 分 → 72)
- 5 类 → ×1.4 (60 分 → 84 / 80 分 → 112 → 100 clamp)
排序：totalSeverity desc → kindCount desc → instrumentID asc（稳定）
```

## 10 · 可能踩的坑（本 session 实测）

### 10.1 macOS 14+ SwiftUI API 不能用（v15.82 切机首暴露）
- Package.swift `.macOS(.v13)` · Linux 编译跳过看不出
- 5 处 `onChange(of:initial:_:) { _, _ in ... }` → 改 `.onChange(of: x) { _ in ... }`
- 文件：SpreadWindow / CalendarSpreadWindow / OptionWindow ×2 / InstrumentDashboardWindow

### 10.2 enum 加 case 后必须扫所有 switch 站点（v15.82 暴露）
- AlertWindow.conditionParams switch 漏 .spreadDeviation case
- 即使有 sheetEditableCases 过滤器 · 表单 switch 仍要 case · 编译器不承认 filter 后的"穷尽"

### 10.3 全局 menu 快捷键防冲突（v15.84 暴露）
- 4 个窗口（Watchlist/Spread/CalendarSpread/Review）内"刷新"按钮都绑了 ⌘R
- 任一为 frontmost 时按 ⌘R 触发"刷新"而不是全局"复盘"
- 修：4 处 ⌘R → ⌘⇧R

### 10.4 bash $VAR 后中文（v15.84+ 暴露）
- `set -u` 模式下 `$BUILD_PATH` 紧跟 `）`（0xEF 0xBC 0x89 UTF-8）
- bash 误把 0xEF 当变量名延续 → BUILD_PATH\xef unbound
- 修：`${BUILD_PATH}` 明确边界

### 10.5 AppleScript 保留字
- `windows` / `window` 是 application 的 every window selector · 不能作变量名
- 修：用 `winList` / `w`

### 10.6 macOS 没装 GNU coreutils
- `timeout 30 cmd` 报 command not found
- 修：用 perl alarm 包装（`perl -e 'alarm shift; exec @ARGV' 30 cmd`）

### 10.7 screencapture -l <window-id> 不可用
- osascript "id of front window" 返 AX accessibility id
- screencapture -l 需要 CGWindowID（要 Quartz Core）· 两者不同
- v15.83 实测走了 fallback 整屏 · 截图仍含 dock / Ghostty
- 暂保留全屏模式 · 后期可单独用 Swift CLI 工具替代

### 10.8 trap EXIT 早期失败应区分（v15.84+ 暴露）
- 脚本启动即失败时 trap 还是会跑 · 不应说"上传完成"
- 修：trap 内加 early_abort 判断（COMPLETED_PHASES = "开始" / "未到 P1"）
- 早期失败：写"🚨 脚本启动即失败"+ 仅传 exit_summary

### 10.9 ComboAnomaly init bonus 命名
- simplifier L1 收尾时建议改：避免嵌套 `1.0 + 0.2 × max(0, kindCount - 3)`
- 命名 `bonus` 表达清晰 · 数学行为不变

### 10.10 mockBars 时间戳不对齐（v15.75 SpreadAlertDetector custom 测试）
- mockBars 用 Date() 调用顺序不同 · baseTime 略有差异（µs 级）
- SpreadCalculator.calculate 严格按 openTime 字典查 · 找不到匹配返 []
- 测试改用 evaluate(values:pair:) 直接路径 · 绕开 mock series 不对齐
