### 5.2 SQLite ALTER TABLE ADD COLUMN 老库 migration 模式（v15.98）

```swift
private func ensureSchema() async throws {
    try await connection.exec("CREATE TABLE IF NOT EXISTS ... (... setup TEXT)")
    // 老库 migration · SQLite 没 ADD COLUMN IF NOT EXISTS · try? 吞重复列错
    try? await connection.exec("ALTER TABLE trades ADD COLUMN setup TEXT;")
}

// bindings 处理 nil → SQL NULL
private func bindings(for t: Trade) -> [SQLiteValue] {
    [..., t.setup.map { .text($0) } ?? .null]
}
```

### 5.4 自动补全 trigger 模式（v16.2）

```swift
// textDidChange 后 dispatch async 检查
public func textDidChange(_ notification: Notification) {
    DispatchQueue.main.async {
        ...
        self.maybeTriggerAutoCompletion(tv)
    }
}

private func maybeTriggerAutoCompletion(_ tv: NSTextView) {
    let range = tv.selectedRange()
    guard range.length == 0, range.location > 0 else { return }
    // 找光标前连续 ASCII 字母（直接 UTF-16 比较避免 String 转换开销）
    var wordStart = range.location
    while wordStart > 0 {
        let codeUnit = ns.character(at: wordStart - 1)
        let isLetter = (codeUnit >= 0x41 && codeUnit <= 0x5A) || (codeUnit >= 0x61 && codeUnit <= 0x7A)
        if isLetter { wordStart -= 1 } else { break }
    }
    guard range.location - wordStart >= 2 else { return }   // 至少 2 字母
    let prefix = ns.substring(...).uppercased()
    guard MaiLangSyntaxHighlighter.allCompletionCandidates.contains(where: { $0.hasPrefix(prefix) }) else { return }
    tv.complete(nil)   // 走 completions(forPartialWordRange:) protocol
}
```

## 10 · 可能踩的坑（本 session 实测）

### 10.1 Direction.rawValue 不是字符串面值（Codable 测试踩过）
- `Direction` rawValue 是 `"0"` / `"1"`（不是 `"buy"` / `"sell"`）
- 写老 JSON 兼容测时手写 JSON 字面量易错 · 改用先 encode 再 decode 间接测兼容性

### 10.2 SQLite ALTER TABLE 没 IF NOT EXISTS
- 标准 SQL 不支持 `ADD COLUMN IF NOT EXISTS`
- 解决：`try?` 吞重复列错（错误 message "duplicate column name"）
- 副作用：列名拼错会沉默吞 · 测覆盖必跑 save/load 真往返

### 10.3 swift test --filter 不识别 swift testing @Suite
- `--filter "SuiteName"` 只匹配 XCTest className · 不匹配 swift testing
- 解决：直接跑全量（10s 左右）· filter 留给 XCTest

### 10.4 Self.pct shadow（v16.3 simplifier 暴露）
- enum 内 `static func pct(_ v: Double) -> String` helper
- 内部局部变量 `let pct = ...` 会 shadow · 编译过但易混淆
- 解决：起独立局部名 `unlabeledPct`

### 10.5 SourceKit 缓存陈旧（多次出现 · 老坑）
- swift build 已 OK · SourceKit 仍报错
- 跑 swift test 实际验证 · 编译过 = 真过

### 10.6 字符串内嵌字面双引号（v16.0 写 SetupEditorSheet hint 提示踩过）
