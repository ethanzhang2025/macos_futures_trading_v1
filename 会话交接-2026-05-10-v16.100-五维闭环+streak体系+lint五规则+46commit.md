### 3.5 Lint 系统 5 规则 + Quick Fix 速查（v16.60/66/74/75/77/78/92/93/95/96/97/98）

```
5 Kind（v16.66 加 undefined / v16.98 加 argCount）：
1. unusedVariable          severity=warning · 中间变量未引用
2. duplicateDefinition     severity=error   · 同名 ≥ 2 次定义
3. missingColorAttribute   severity=warning · 输出未 COLOR
4. undefinedVariable       severity=error   · 标识符未定义（含 typo 建议）
5. argCountMismatch        severity=error   · 函数参数数量与签名不符（v16.98）

Severity（v16.74）：
- enum Severity: warning / error · Comparable（warning < error）
- Kind.defaultSeverity 派生

Quick Fix Menu（v16.60/97/98）：
- unused → 删除该行
- duplicate → 跳到首次定义 / 删除重复行
- missingColor → 子菜单插入 COLOR* (6 选)
- undefined → 跳转检查 + 复制 typo 建议名（v16.97）
- argCount → 跳转 + 复制签名模板（v16.98）

5 处展现：

4. 行内波浪线（按 severity 颜色 · v16.92 + split 同步 v16.96）
5. quick fix Menu（v16.60/97/98）

typo 建议（v16.93/95/97）：
- closestKnownName · Levenshtein 距离 ≤ 2 + 长度差 ≤ 2
```

### 10.2 测试 flaky（与 v16.54 同 · 与本 session 无关）
- ReplayDriver Timer 偶发 issue · re-run 通过
- CalendarSpread "contango 主导" 阈值 70% 偶尔 flaky · re-run 通过
- 已记入永久记忆 · 不再深究

### 10.3 -0.0 IEEE 符号污染 CSV（v16.61 暴露）
- `max(0, -(pairPnLs.min() ?? 0))` 当 min 为 0 时返回 -0.0
- `String(format: "%.2f", -0.0)` = "-0.00"（CSV 输出脏）
- 修：`worstLossRaw < 0 ? -worstLossRaw : 0` 显式 normalize 符号

### 10.4 split() omittingEmptySubsequences 默认 true（v16.61 测试暴露）
- `lines[1].split(separator: ",")` 默认吞空字段
- 空 pattern 列被吞导致 index 偏移测试失败
- 修：`.split(separator: ",", omittingEmptySubsequences: false)`

### 10.5 outline 每行只取首个语句（v16.66 测试暴露）
- MaiLangOutline.parse 每行 trim 后找首个 `:` / `:=`
- `MA5:=...;OUT:...` 一行只识别 MA5（OUT 不被识别）
- 测试 src 要每语句独立一行

### 10.6 closestKnownName 短词 < 3 不建议（v16.93 边界）
- "MA" 2 字符 → 直接返回 nil（防噪音）
- 测试用 "MAA"（3 字符）+ "MAAB"/"MAAC" 验证平局取最短

### 10.7 Mac swift 6.3 严格 concurrency（v16.30 永久记忆 · 仍待验）
