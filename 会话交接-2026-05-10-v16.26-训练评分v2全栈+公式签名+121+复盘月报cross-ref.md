## 5 · 关键设计沉淀（本 session 新增）

### 5.1 v2 五维评分系统（v16.6 · 不破坏 v1 历史兼容）

```
设计核心：
- v1 totalScore 算法不变（pnlScore + disciplineScore = 0-100）· 保持老历史稳定
- v2 subScores 仅作分析视角 · 不参与总分加和
- TrainingSubScores: { pnl/discipline/winRate/risk/efficiency: Int 0-100 + weakest + weakness }
- subScores 字段为 Optional · Codable 自动合成 decode/encodeIfPresent · 老 JSON 反序列化 = nil

5 维计算（纯函数）：
- pnl/discipline · v1 主分 ×2 派生（数学等价 · 单一真源 · simplifier 第 1 轮去重）
- winRate/risk/efficiency · 由 trades FIFO 配对（合约+方向）派生 ClosedPair 算
  · winRate · pairs 中 pnl > 0 占比 ×100 · 无 pair → 50 中性
  · risk · 单笔最大亏损率 5% → 0 / 0% → 100 线性
  · efficiency · 平均每笔 pnl% · ±0.5% → 0/100 线性

weakness 选最弱：
- min(by: { $0.score < $1.score }) · 同分时落 enum first 顺序（pnl 优先 · 隐性约定 · 测试依赖）
- weaknessAdvice · 5 条中文改进建议
```

### 5.2 同形态历史对比（v16.13 · trader 改进可视化）

```
PatternComparison 数据模型（不依赖 ClosedPosition · 仅 TrainingSessionLog）：
- patternComparison(for: sessionID) → 找出该 pattern 历史 prior（不含当前 session）
- Trend 阈值：±3 分（< 3 = flat · ≥ 3 = up/down · priorCount=0 = firstTime）
- isNewBest = (priorCount > 0 && currentScore > priorBestScore)

UI（patternComparisonStrip · TrainingScoreSheet 头部下方）：
- 形态名 + 历史次数 + 均分 + ↑↓→✨ emoji + delta（+12/-7/持平）+ 🏆 新高
- trendColor: up=green / down=red / flat=secondary / firstTime=blue
```

### 5.3 SwiftUI Canvas 几何（v16.14 雷达图 + v16.16 sparkline）

```
雷达图（130×130）：
- 5 顶点 cos/sin 极坐标 · startAngle = -π/2（顶部）· 顺时针 angleStep = 2π/5
- 4 圈参考（背景 + 25/50/75）+ 5 中心射线
- 实际分多边形：ratio = score / 100.0 · fill 蓝 0.18 + stroke 蓝 1.5pt
- 顶点圆点 · 最弱橙色加粗 r=3.5（其他 r=2.5）
- emoji 外置标签 · ratio = 1.0 + 14/maxR

Sparkline（横向折线 · 36pt 高）：
- 30 次按 endedAt 升序 · xStep = width / (n-1)
- yFor(score): height - 2 - (height-4) × score/100（边距 2pt）
- 70 分参考线虚线（dash 2,2）· 折线 stroke accentColor 1.5pt
- 端点圆点 · 最后笔 r=3 加粗（其他 r=1.5）· trendDeltaChip（最近 N 笔 vs 之前均值）
```

### 5.5 CSV 导入 / 导出对称模式（v16.20/v16.24）

```
Exporter（与 TradeCSVExporter / ClosedPositionCSVExporter 同模式）：
- UTF-8 BOM + CRLF · RFC 4180 · escape 含逗号引号字段
- 时间降序（最新在前 · trader 习惯）
- 20 列：v1 主分 + v2 五维 + 违规计数 + 交易笔数
- subScores nil → 空字段（不写 "nil"）

Importer（容错优先）：
- 容忍 BOM 前缀 + LF/CRLF · header 中文标题查列索引
- 关键 5 列必填（时间/时长/场景/资金前后）· 其他 nil-tolerant
- 无效行（时间格式错 / 数值无法 parse）静默跳过 · 不抛
- header 缺关键列 → 整体返回空（保护性）
- violations 重建 severity + count（详细 message 不还原 · 评分仅用 count）
- pattern 中文 displayName 反查 enum

CSV 解析（RFC 4180 简化）：
- 字段以 , 分隔 · " 包围允许内嵌 , / \n · "" 转义为 "
```

### 10.4 NSTextView layoutManager / textContainerOrigin 几何
- glyphIndex(for:in:) 入参是 textContainer 局部坐标（要先减 textContainerOrigin）

### 10.7 NSEvent.locationInWindow vs view-local 坐标
- mouseMoved 事件的 locationInWindow 是 window 坐标
