## 5 · 关键设计沉淀（本 session 新增）

### 5.1 Heikin Ashi 变换设计（v17.13）

```
设计：HA 是 trader 看趋势利器（去噪 · 同色 candle 串）。
仅替换 candle 渲染层 bars · HUD/hover/indicators 仍用原始 OHLC（保正确性）。
KLine.heikinAshi(from:) static func · 公式 5 行：
  HA_close = (O+H+L+C) / 4
  HA_open[0] = (O[0] + C[0]) / 2
  HA_open[i] = (prev_HA_open + prev_HA_close) / 2
  HA_high = max(H, HA_open, HA_close)
  HA_low = min(L, HA_open, HA_close)
volume / openInterest / turnover 保留。
ChartType enum · UserDefaults chartType.v1 · 仿 ChartTheme 模式。
```

### 5.2 通道线 regression（v17.11）

```
设计：用户给 2 点定 [startBar, endBar] range · 内部对 bars[range].close 做最小二乘。
DrawingGeometry.channelRegression(closes:) -> (slope, intercept, stdDev)
渲染 3 线：主线（实线）+ 上轨 main+1σ（虚线）+ 下轨 main-1σ（虚线）+ 8% 填充。
Hit test 同 3 线段最小距离。
```

### 5.3 斐波 6 项扩展（v17.16/17/18）

```
设计共性：复用 FibonacciLevels.standard / projection / fanCore 常量。
- 斐波扩展：levels 改为 > 1.0（externRation 6 个）
- 斐波弧：屏幕距离半圆（避免数据空间椭圆）· Path.addArc
- 斐波通道：startPoint + endPoint + channelOffset · 内部按 standard 7 ratios 偏移

A 段斐波系列共 6 种（fib/extension/arc/channel/fan/timeZone）· 国内首家全套。
```

### 5.4 Shell 真数据集成模式（v17.21/24）

```
设计共性：UserDefaults didChangeNotification 跨窗口同步。
- @State 接 store.load() 初始化
- .onReceive UserDefaults didChangeNotification { snap = store.load() } 实时跟随
- TradingWindow 改持仓 → Sidebar + BottomTradingBar 立即刷新（不用 EventBus）
- 性能：UserDefaults didChange 触发频率不高 · load 是 JSON 解码（< 1ms）· 可接受
```
