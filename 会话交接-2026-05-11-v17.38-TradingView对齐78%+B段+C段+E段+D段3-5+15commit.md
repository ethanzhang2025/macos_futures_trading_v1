## 5 · 工程踩坑（本 session 实测）

### 5.1 SourceKit 缓存陈旧（已知 · v17.29 9.1 同）
- 新加 enum case / struct member 后 SourceKit 报"no member" · build/test 实际通过
- 多次出现：trendLineCrossed / valueArea / spread / webhook / colorIndex / setGroupColor / InstrumentFlag
- 忽略即可

### 5.2 actor Sendable 限制
- 新 struct 含 UserDefaults 不能 `: Sendable`（UserDefaults 是 reference type · 非 Sendable）
- 解：去掉 `: Sendable`（InstrumentFlagStore）

### 5.3 #expect 不能直接调 mutating method
- `#expect(book.setGroupColor(...))` 失败：closure 内 `$0` 是 immutable
- 解：先 `let r = book.setX(); #expect(r)` 拆开

### 5.4 try #require 需要 throws function
- 否则报"No calls to throwing functions occur within 'try' expression"
- 解：把 test func 标 `throws`

### 5.5 actor exhaustive switch 同步
- 加 NotificationChannelKind 新 case `.webhook` · 必须同步 AlertConfigCSVExporter / AlertHistoryCSVExporter / AlertHistoryFilter 的 switch
- 加 AlertCondition 新 case `.trendLineCrossed` · 同上

### 5.6 Watchlist Codable 兼容旧 JSON
- 加 `colorIndex` 字段 · 必须 `decodeIfPresent` + `encodeIfPresent`（与 version/deletedAt 同模式）

### 5.7 Lexer/Parser init 标签
- 不是 `Lexer(text:)` · 是 `Lexer(source:)`
- Lexer/Parser 都需 `var`（不是 `let`）· mutating tokenize/parse

### 5.8 CalendarSpreadCalculator contango flaky（已知）
- 概率 mock 0.695 vs 0.7 阈值 · 偶发失败 · 重跑通过
- 加入 flaky 列表（与 ReplayDriver Timer / WP-44b 同列）
