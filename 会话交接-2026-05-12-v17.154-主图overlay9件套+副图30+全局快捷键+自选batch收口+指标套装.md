## 10 · 本 session 学到的（写入下次会话提醒）

### 模块层 + UI 层分两 commit 收口（v17.145/146 archive 模式）

archive batch 三步：
- v17.145 模型层 + 11 单测（决策稳定 · 单 commit 收口）
- v17.146 UI 行/右键 接通
- v17.147 toolbar 显隐 toggle 收尾

每 commit 范围聚焦 · 模型测试覆盖完后 UI 改动风险面小 · 不必再加大量 UI 单测。

### 9 字段 Codable 阶梯式 fallback 守（v17.139/140/152/153 模板）

每加新字段 commit 都加：
1. CodingKeys case 加新 key
2. decodeIfPresent ?? 默认值
3. 单测加 `oldVx_yJSONFallback` 用 `vx_y 时构造的 book` encode 当 oldJSON · decode 验新字段全 fallback + 旧字段保留

每个 v17.xxx 加字段 commit 末加一个 oldVxxxJSONFallback 单测 · 跨 4 commit 阶梯式覆盖（v17.139/140/152 模板）。

### onChange 统一 save + 重算（v17.151 重构教训）

v17.139 menu toggle 写 `setEnabled + save + Task updateIndicatorsFull` 三步重复在每个 set closure。
v17.151 重构改为 `set: { newVal in overlayBook.setEnabled(kind, newVal) }` + `.onChange(of: overlayBook) { save + Task }` 统一处理。

好处：toggle 和 sheet 保存 走同一路径 · 防遗漏 + 单点维护。新 menu 改 binding set 时只需 mutate state · onChange 自动接管。

### IndicatorCore 算法已有则直接复用（v17.139/140/142/143/144/153）

每加新 overlay/副图前先 grep IndicatorCore 已有算法。如：
- VWAP / PivotPoints 已存在 → 删我重复 · 复用现成
- Ichimoku / Donchian / KC / SAR / PriceChannel / Envelopes / TRIX / CMF / PVT / VR / ADX / ADL / MFI / CMO 全已有
- 唯一新 IndicatorCore 实装：SuperTrend（与麦语言 SUPERTRENDFunction 算法对齐）

不重复实现 IndicatorCore 算法 = 单源真相 + 麦语言/主图/副图三处共享同一逻辑。

### Pivot 7→5 线只在 ChartScene compute 过滤（v17.150 决策）

不动 IndicatorCore PivotPoints.calculate 输出（仍 7 列 · 麦语言 BUILTIN ICHIPIVOT 等用全量）· 只在 MainChartOverlayCompute 过滤 R3/S3。

好处：trader UI 紧凑 + IndicatorCore 算法稳定 · 不破坏其他调用方。

### Mac swift 6.3 严格 Decimal 字面量（v17.153 教训）

`init` 默认参数 `Decimal = 0.02` Linux 编译过 · Mac 严格模式可能报 type-checking 慢。

写法用 `Decimal(string: "0.02") ?? Decimal(0.02)`（防 nil + Linux 兼容）。本 session SAR/Envelopes 默认参数都用此模板。

### 字符串 raw 跨 module 解耦（v17.154 IndicatorPreset 模式）

Shared 不能引 MainApp 类型（SubIndicatorKind 在 MainApp）· 用 rawValue 字符串引用：
- IndicatorPreset.subIndicatorRaws: [String]
- 调用方 `compactMap(SubIndicatorKind.init(rawValue:))` 映射枚举

trade-off：失去类型检查 · 但解决 module 依赖循环。新 SubIndicatorKind case 加上后 preset raw 不会自动同步 · 加新 case 时记得检查 preset 是否要包含。
