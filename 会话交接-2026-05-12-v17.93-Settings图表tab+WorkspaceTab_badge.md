## 4 · 本 session 关键产出

### A · Settings 图表 tab 实装（v17.92）

**关键决策**：**不重复造轮**。

探查后发现 `Sources/Shared/Models/IndicatorParamsBook.swift` 已是完整指标参数系统（19 类参数 · UserDefaults `indicators.params.v1` 持久化 · 已被 ChartScene 使用）。Settings 图表 tab 直接复用 `IndicatorParamsStore` 编辑这个 book 即可。

**新建 `Sources/Shared/Models/ChartSettingsStore.swift`** 补 IndicatorParamsBook 没覆盖的两块：
- `enum CandleColorMode`：`.redUpGreenDown`（中国默认）/ `.greenUpRedDown`（国际/TradingView）
- `enum PricePrecisionMode`：`.auto`（跟随合约）/ `.fixed2` / `.fixed3` / `.fixed4` · `digits: Int?` computed
- `load / save / resetAll` · UserDefaults keys `chart.settings.v1.candleColorMode` + `chart.settings.v1.pricePrecision`

**Stubs.swift 改造**：
- 删除原 line 23 placeholder · 替换为 `ChartSettingsTab()`
- 新 `private struct ChartSettingsTab` · Form + 4 Section + intStepper helper
  - **主图指标默认**：MA 3 周期 / BOLL period + σ multiple
  - **副图指标默认**：MACD（fast/slow/signal）/ KDJ（period/smoothK/smoothD）/ RSI period
  - **K 线配色**：CandleColorMode 单选 Menu
  - **价格显示**：PricePrecisionMode 单选 Menu
  - **恢复全部默认**：红色 Button · NSAlert 二次确认 · 重置 `IndicatorParamsStore.save(.default)` + `ChartSettingsStore.resetAll()`
- `onChange` 自动持久化（save 不阻塞 form 输入）

**测试**：`Tests/SharedTests/ChartSettingsStoreTests.swift` · 7 case：
- 默认 candle color = redUpGreenDown
- 默认 price precision = auto
- candle color round-trip
- price precision 全 4 选项 round-trip
- digits computed（auto = nil / fixed = 2/3/4）
- 非法 rawValue 回退默认
- resetAll 清空两 key 回到默认

**关键 ⚠️ 未完成**：
- ChartScene 仍 hardcoded 用 `ChartTheme.chartProfit/.chartLoss`（绿/红 固定）· **未读 CandleColorMode** · 用户在 Settings 切换"涨绿跌红"后 ChartScene 不响应
- 价格 label 仍 hardcoded 用合约 stepSize 推算 · **未读 PricePrecisionMode** · 用户固定 N 位时无效
- 设计意图：本次只交付 "用户可调 + 持久化" 层 · ChartScene 接通留后续小 commit 推进（接续推荐 🥇）
