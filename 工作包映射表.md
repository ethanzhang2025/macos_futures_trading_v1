## 编码开工推荐流程

当要进入某 WP 实际编码时：

1. 读我的 `Stage A 工作包清单.md` / `Stage B 工作包清单.md` 找到 WP-XX（取时点/负责/DoD/锚点/禁做项）
2. 查本映射表找 ChatGPT 对应包（如 A03）
3. 读 `chatgpt_工作包清单/stage-a/A03-Metal 图表引擎 v1.md` 取启动 prompt + 范围 Out + 禁做项 + 验收
4. 合并 1 + 3 信息喂入 Claude Code 新会话或子代理
5. 完成后更新**我的清单**状态（⬜ → 🟨 → ✅）；ChatGPT 目录状态不维护

如 WP 无 ChatGPT 对应（非工程），直接用我的清单 + `Claude Code 启动 prompt 模板.md`。
