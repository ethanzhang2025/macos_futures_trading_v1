**实装方案备忘**（开工时直接套）：
- 数据模型：`enum PaneLayout` → `indirect enum PaneNode { case leaf(PaneConfig); case hSplit([PaneNode], ratios: [Double]); case vSplit(...) }`
- 渲染：AppKit `NSSplitView` + delegate（autosaveName / 自定义分隔条 / 嵌套 split）· 通过 `NSViewRepresentable` 桥接 SwiftUI
- 拖拽：Pane 标题栏长按 → `onDrag` → 落点检测（4 边或中心）→ 重组树
- 持久化：递归 Codable 树结构存到 Workspace JSON · 兼容旧 enum schema（迁移读取）
- 测试覆盖：树结构 Codable 双向 / 7 预设 → tree 等价性 / 拖拽落点逻辑
