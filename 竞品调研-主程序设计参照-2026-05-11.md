# 竞品调研 · 主程序设计参照

> **目的**：为"中国期货 Mac/iPad 原生工作台"设计主程序 Shell 架构（取代 28 独立窗口）
> **基准时点**：v16.215 · 2026-05-11
> **范围**：14 个核心竞品 × 10 维度 · 3 标杆深度 + 11 个广度
> **使用**：Shell 架构设计前必读 · 后续 v17.0 设计文档基础

---

## 0 · 执行摘要

**关键结论**：

1. **所有 14 个竞品都已放弃"多独立窗口"模式**——这正是我们 28 窗口的痛点。主流是**单 Shell 容器 + 多 Tab/Page + 组件切分**。
2. **没有一个竞品是 macOS 原生**（SwiftUI/AppKit/Metal）——这是我们最大的差异化护城河。
3. **彩色 group 同合约联动**（IBKR / 富牛）是国内期货软件普遍空白 · 直接超车点。
4. **底部交易区常驻**是期货软件区别于股票软件的核心特征。
5. **Workspace 命名保存 + 多屏切换**是国外专业软件标配，国内股票期货软件做得明显薄。
6. **文华 F 键体系**是国内期货 trader 肌肉记忆 · 必须兼容。
7. **⌘K 命令面板**（Bloomberg GO 现代化版）是 14 个产品中无人做的现代化机会。

## 1 · 标杆 1：富途牛牛 macOS 版（"富牛"）

> 国内 trader 看美股/港股首选 macOS app · 单窗口承载、组件+tab 二级嵌套

### 主程序布局

```
┌────────────────────────────────────────────────────────────────────┐
│ [LOGO][搜索]  [自选][行情][交易][社区][发现][复盘]    [资产][消息] │
├──────┬──────────────────────────────────────────────┬──────────────┤
│自选   │  K 线主图（默认主区 · 1 chart 全宽）         │ 盘口 5/10 档 │
│板块树 │                                               │ 分时          │
│ AH股 │  ┌─────────────┐ ┌─────────────┐              │ Tick 明细     │
│ 港股 │  │ 副图 MACD   │ │ 副图 RSI    │              │ 资讯/异动     │
│ 美股 │  └─────────────┘ └─────────────┘              │              │
│ 期货 │  分时/周期/复权/叠加                          │ 自定义组件槽  │
├──────┴──────────────────────────────────────────────┴──────────────┤
│ 持仓 | 委托 | 成交 | 条件单 | 资金 | 算法单            ← 底部交易抽屉 │
└────────────────────────────────────────────────────────────────────┘
```

**区域比例**：左 sidebar 12-18% · 主区 55-65% · 右盘口 18-22% · 底部抽屉默认收起

### 3 层 Tab 模型（富牛的"秘密武器"）

1. **顶部一级 tab**（自选/行情/交易/社区/发现）—— 大功能模块
2. **二级 tab**（一级内）—— 用户自命名"工作台快照"
3. **tab 内组件槽** —— 25+ 种组件（K 线/盘口/Tick/新闻/龙虎榜/异动池/资金流/热力图）可拖拽分割

**实际 trader 用法**：
- tab "早盘开盘"：4 个合约 K 线 + 行业热力图
- tab "日内主战场"：1 大 K + 盘口 + Tick + 持仓 + 条件单
- tab "夜盘"：低对比深灰 + 组件密度高
- tab "复盘"：K 线 + 笔记 + 新闻流

### ⭐ 杀手锏：12 色分组联动

- 最多 **12 组**，每组一个颜色编号
- 同组所有组件（不论在哪 tab/哪屏）**同步切换标的**
- 单台机器可监控 48+ 标的

### 多屏 / 分离窗口

- tab 可**拖出主窗口**变独立窗口
- 拖回又合并
- 独立窗口仍受分组联动控制
- 工作台模板可命名保存（云端同步）

### 快捷键

- ⌘+数字 1-9 切顶部一级 tab
- ⌘+T 新建二级 tab · ⌘+W 关闭
- ⌘+F 全局搜（合约/新闻/帖子统一入口）

### 视觉 / macOS 适配

- 默认红涨绿跌（中国习惯）· 可切美股蓝/红
- 涨跌停加粗 + 高亮背景
- 主力合约自动切换
- **不是 macOS 原生 · Qt + web 组件混合**（启动慢 5s+ · 内存 1.5GB+）
- 但 Mac/Win 体验一致 · trader 设备切换无认知负担

## 2 · 标杆 2：文华财经赢顺 wh6 / 赢智 wh9

> 国内期货 trader 第一选择 · 19 年沉淀 · 报价表为中心 + F 键直跳

### 主程序布局

```
┌──────────────────────────────────────────────────────────────────────┐
│ 文件 视图 报价 分析 交易 个性化 帮助    [合约搜索]   [账户][登录]   │
├────────┬────────────────────────────────────────────────────────────┤
│ F6自选 │   报价列表（默认主区）：                                    │
│ F2商指 │   ┌──────────────────────────────────────────────┐         │
│ F3上证 │   │ 合约 | 现价 | 涨幅 | 持仓 | 成交 | 主力        │         │
│ F4深证 │   │ rb2601 ... ru2601 ...                          │         │
│        │   └──────────────────────────────────────────────┘         │
│ 黑色   │   Enter → 进入 K 线大图（按 Esc 返回）                      │
│ 有色   │                                                              │
│ ...    │                                                              │
├────────┴────────────────────────────────────────────────────────────┤
│ [空格] 唤起下单主窗口（浮动）  [Shift+~] 盘口炒单面板                │
└──────────────────────────────────────────────────────────────────────┘
```

### UI 哲学：**Enter/Esc 二态切换**

- 进入即看报价
- Enter → K 线大图
- Esc → 回报价
- **所有操作围绕这两态**

### 自设页面（≈ tab 概念）

- 路径：右上 `个性化 → 新建页面`
- 一个页面 = 任意 N×M 分割窗口（2x2 / 1+2 / 3x3 常见）
- 每窗口 = K 线/分时/盘口/报价表/Tick
- 窗口左上**联动按钮**：开启后同步切合约
- **跨周期十字光标联动**（Alt+V）：分钟图 + 日线 + 周线十字线对齐到同一时刻 · 多周期共振分析

### ⭐ F 键体系（国内期货 trader 的"标准 IME"）

| 键 | 功能 |
|---|---|
| **F6** | 自选合约 |
| **F8 / Ctrl+F8** | 周期循环切换（1m→3m→5m→15m→...→年线 / 反向）|
| **F12** | 画线工具 |
| **F10** | 基本面 / 合约资料 |
| **F11** | 下载历史 K 线 |
| **F2/F3/F4** | 商品指数 / 上证 / 深证 |
| **Enter** | 报价表 → K 线大图 |
| **Esc** | 返回报价 / 窗口还原 |
| **空格** | **唤起下单主窗口**（最高频快捷键）|
| **Shift+~** | 盘口炒单面板（剥头皮必用）|
| **Ctrl+L** | 显示/隐藏盘口 |
| **Ctrl+W** | 全屏 |
| **Alt+K** | 切换复权 |
| **Alt+V** | 跨周期十字光标联动 |
| **数字键** | 报价排序（60 原序 / 61 涨跌幅 / 62 成交量 / 900 主力排名）|
| **方向键** | K 线缩放与翻页 |

**这是国内期货 trader 90% 必须兼容的肌肉记忆**

### 视觉 / macOS 适配

- 红涨绿跌（**不可切**）
- 信息密度极高（一屏 30+ 合约 + 多 K 线无压力）
- 字号小、行距密
- 暗黑模式有但 trader 多用浅灰背景
- 主力合约自动换月（rb 跟随持仓量最大）
- 涨跌停加亮
- **麦语言公式**（核心战略资产 · 200+ 函数 · 零编程门槛）
- **Mac 版是 Win 移植 · 非 macOS 原生** · 但 trader 觉得"亲切"

## 3 · 标杆 3：TradingView Desktop

> Chart 至上 + Workspace 即 Layout · Electron · macOS 适配最好

### 主程序布局

```
┌──────────────────────────────────────────────────────────────────────┐
│ [chart] [chart 2] [+ 新tab]    Layout: My Setup ▾   ⚙ ⌘+P  👤        │
├──────┬──────────────────────────────────────────────┬────────────────┤
│左工具 │   多 chart 分屏（1 / 2 / 4 / 6 / 8 配置）    │ 右侧侧栏       │
│Cursor│   ┌──────────┐ ┌──────────┐                  │ ├─ Watchlist  │
│趋势线│   │  Chart 1 │ │ Chart 2  │                  │ ├─ Details    │
│水平线│   │  CL 1H   │ │ CL 1D    │                  │ ├─ News       │
│Fib   │   ├──────────┤ ├──────────┤                  │ ├─ Hotlist    │
│形态  │   │  Chart 3 │ │ Chart 4  │                  │ ├─ Calendar   │
│文本  │   │  CL 1W   │ │ CL 4H    │                  │ ├─ Ideas      │
│魔棒  │   └──────────┘ └──────────┘                  │ ├─ DOM        │
├──────┴──────────────────────────────────────────────┴────────────────┤
│ 底部时间轴 / 复制图层 / 重放（Replay）                                │
└──────────────────────────────────────────────────────────────────────┘
```

### ⭐ Layout vs Tab 分离

- **Tab**（Chrome 标签栏）：上下文容器
- **Layout**：tab 内的具体布局快照（指标 / 绘图 / 设置全保存）
- **多 chart 分屏：1/2/4/6/8** · 单 tab 最多 8 chart
- **Tab linking by symbol**：链接的 tab 共享 symbol（按颜色组联动）
- **Crosshair 同步**：跨 chart / 跨 tab / 跨 window 十字光标对齐

### 多屏

- ⌘+N 开新窗口（同账号同步 layout）
- 任何 tab 可拖出独立窗口
- **多屏 workspace 可保存恢复**
- Layout / watchlist / 设置全部走云端账号同步

### ⌘ 体系（最 macOS 化）

| 快捷键 | 功能 |
|--------|------|
| **⌘+T** | 新 tab |
| **⌘+U** | 复制当前 tab |
| **⌘+N** | 新窗口 |
| **⌘+W** | 关闭 tab |
| **⌘+Shift+T** | 恢复关闭的 tab |
| **⌘+1..8** | 跳到指定 tab |
| **⌘+9** | 跳到最右 tab |
| **⌘+R** | 刷新 |
| **⌘+Q** | 关 app 保留布局（下次回来一模一样）|
| **⌘+,** | 偏好设置 |
| **Tab** | 多 chart layout 内切 chart |
| **Alt+Enter** | 单 chart 全屏 |

画线工具用字母键：T 趋势线、H 水平线、V 垂直线、F Fib、A 箭头

### 视觉 / macOS 适配

- 默认绿涨红跌（西方）· 可切红涨绿跌
- 多种自定义配色（任何国内软件都比不过）
- **Electron 包装 web · 非 SwiftUI** · 启动 2-3 秒 · 内存 800MB-1.2GB
- macOS 原生菜单栏 · ⌘ 体系 · ⌘+Q 保留状态
- 触摸板二指缩放 chart · 暗黑跟随系统
- 无 Quick Look / Stage Manager / Continuity Camera 深度适配

### 4.3 国内共同模式（提炼）

**期货软件共同**：
1. **5 区布局 + 底部交易区**：90% 是"顶部菜单 + 左合约树 + 中主区 + 右辅助 + **底部固定交易区**"
2. **"页面 Page"是统一抽象**：博易"新建页面 + 切分"、极星"Page Tab + 切分"、文华"组合窗口"——本质都是"一 Page = 一组切分区域"
3. **多账号原生**：持仓栏 dropdown 多账号切换
4. **套利专项页**：极星"本地套利" · 跨合约价差独立 Page

**股票软件共同**：
1. **大主窗口 + 副屏**：同花顺副屏最完整 · 通达信副屏最弱
2. **快捷键文化**：F5/F8/F9/Alt+Z 是肌肉记忆
3. **多股同列**：4/6/9 宫格"分时同列 / K 线同列"
4. **F10/F9 单键弹窗**：详情用弹窗而非新 tab

### 4.4 国内 macOS 端真相

| 软件 | Mac 端状态 | 评价 |
|------|-----------|------|
| **易盛极星 9.5** | ✅ 原生 Mac 版 | 国内期货软件 Mac 端唯一可用 · 不支持程序化 |
| **同花顺股票版** | ✅ 原生 Mac 版 | 国内股票软件 Mac 端最好 · macOS 风 · 多券商 · 极端行情 1-2s 延迟 |
| **东方财富 Choice** | ✅ v8 全平台 | 终端类 Mac 投入最积极 |
| 通达信 | ⚠️ Mac v3.59 | 简陋到无法直视 · 缺编程 · 社区不推荐 |
| 大智慧 | ⚠️ Mac v3 | 视觉老旧 · 功能简化 |
| 博易大师 | ❌ 无 Mac | 只有 iOS + Windows |
| 文华财经 | ⚠️ Mac 移植 | Win UI 强搬 · trader 觉得亲切 |
| 涨乐 / e 海通财 / 国君富易 | ❌ 无 Mac | 只有 Win + 移动 · Mac 用户走模拟器 |

**核心痛点**（来自社区）：
- 多数 Mac 版是 Win UI 强搬 · 不符合 macOS 视觉规范
- 无多屏 + 布局记忆（专业 trader 工作站底线）
- Mac 编程/插件生态缺失（通达信公式、文华麦语言在 Mac 无解）
- 延迟与稳定性不如 Win

## 5 · 国外 6 个广度对照

| 软件 | 主架构 | Workspace | macOS 原生度 |
|------|--------|-----------|-------------|
| **ThinkOrSwim** | 单主窗口 + 顶部 8 核心 Tab + 左 Gadget Sidebar | 命名保存 · 整体切换 · 预设 Forex/Options/ActiveTrader 模板 | Java Swing · Apple Silicon native 后启动 18s · **非 AppKit** |

| **MetaTrader 5** | MDI 单主窗 + 左 MarketWatch/Navigator + 下 Toolbox | Profile = 整体 workspace · Template = 单图样式 | ⚠️ Wine + Rosetta 2 双层翻译 · "假 native" |

### 5.1 国外共同模式

1. **Workspace 命名保存** 是标配（中国期货软件做得明显薄）
2. **主壳 + 浮窗双层**：管理面 + 交易窗口 detach
3. **底部 Tab 切 Workspace**：IBKR/Webull 最直观
4. ⭐ **Linked symbol propagation（同 ticker 联动）**：IBKR 7 色 group 是教科书示例
5. **Detach to floating window** 是多屏唯一通路

### 5.2 macOS 原生程度排名

| 产品 | macOS 原生 | 渲染层 | Apple Silicon | 体验 |
|------|----------|--------|--------------|------|
| Bloomberg Terminal | ❌ | 仅 Win/浏览器 | N/A | 1/10 |
| NinjaTrader 8 | ❌ | WPF | 需 Parallels | 1/10 |
| MetaTrader 5 | ⚠️ "假" | Wine + Rosetta 2 | 双层翻译 | 3/10 |
| ThinkOrSwim | ⚠️ Java | JVM | ARM JDK | 5/10 |
| IBKR TWS | ⚠️ Java | JVM | 单独 ARM 安装包 | 5/10 |
| Webull Desktop | ⚠️ Electron | Web 渲染 | ARM | 6/10 |
| **我们项目（v17.0+）** | ✅ **SwiftUI/AppKit/Metal** | **原生** | **原生** | **目标 9/10** |

**结论**：**14 个竞品没有一个使用 AppKit/SwiftUI/Metal 原生路径**。Mac 原生体验是绝对的空白市场。

## 6 · 14 个产品共同设计模式（综合提炼）

### 6.1 主程序架构（10 项共同模式）

| # | 模式 | 出现产品 |
|---|------|---------|
| 1 | **单 Shell 容器 + 多组件分屏** | 全部 14 个（取代多独立窗口）|
| 2 | **多 Tab/Page 模型**（用户自命名工作台）| 富牛 / 文华 / TV / IBKR / Webull / 极星 |
| 3 | **组件协议**（PaneProtocol · 可拖拽/分组/独立成窗）| 富牛 / TV / IBKR / 极星 |
| 4 | **底部 Tab 切 Workspace** | IBKR / Webull / NT8 |
| 5 | **Workspace 命名保存 + 多屏分配** | 全部国外 6 个 + 极星 + 东财 |
| 6 | ⭐ **彩色 group 同合约联动** | 富牛（12 色）/ IBKR（7 色）/ TV |
| 7 | **跨周期十字光标同步** | 文华（Alt+V）/ TV |

## 9 · 一句话结论

**14 个竞品全部停留在"跨平台/Windows 移植/Java/Electron 的视觉割裂"阶段——没有一个把 macOS 当一等公民。**

我们用 **SwiftUI + AppKit + Metal 原生**路线，在以下 6 个具体点结构性碾压：

1. **macOS 原生视觉**（vibrancy / SF Mono / 系统色 token / 暗黑跟随）
2. **Workspace 多屏 + Stage Manager 集成**
3. **彩色 group 同合约联动**（IBKR + 富牛融合）
4. **F 键肌肉记忆兼容**（文华）+ ⌘ 体系（macOS）双轨
5. **⌘K 命令面板**（Bloomberg 现代化版 · 国内国外首家）
6. **训练评分 + 改进 plan + 套利 + 中国期货专精**（v15-v16 独家护城河）

**不是做更多功能，而是把已有功能用 Mac 应该有的样子做一遍。**
