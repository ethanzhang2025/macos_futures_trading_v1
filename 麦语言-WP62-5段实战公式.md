## 1 · 段 1 — HT_TRENDLINE 趋势过滤系统

**含义**：Hilbert 变换提取 instantaneous 趋势线 + ATR(14) 止损带 + 收盘价上穿趋势线 = 多头信号。

**公式**：
```
TREND:HT_TRENDLINE(CLOSE);
ATRV:ATR(14);
LONG_STOP:TREND-2*ATRV;
SHORT_STOP:TREND+2*ATRV;
SIGNAL:CROSS(CLOSE,TREND);
```

**预期看到**：
- 主图：CLOSE 蜡烛 + TREND 中线（平滑的趋势）+ LONG_STOP / SHORT_STOP 上下止损带
- 副图：SIGNAL 在 CLOSE 上穿 TREND 时跳到 1，其余为 0
- 止损带应正确包夹 TREND（LONG_STOP < TREND < SHORT_STOP）

---

## 2 · 段 2 — KELLY 凯利仓位管理

**含义**：基于最近 60 根 K 线的胜率（COUNT）+ 平均盈/亏（SMA + IF + REF），动态计算凯利建议仓位比例。

**公式**：
```
WIN_BAR:CLOSE>REF(CLOSE,1);
WIN_RATE:COUNT(WIN_BAR,60)/60;
AVG_WIN:SMA(IF(WIN_BAR,CLOSE-REF(CLOSE,1),0),60,1);
AVG_LOSS:SMA(IF(CLOSE<REF(CLOSE,1),REF(CLOSE,1)-CLOSE,0),60,1);
POS:KELLY(WIN_RATE,AVG_WIN,AVG_LOSS);
```

**预期看到**：
- WIN_RATE：副图 0~1 区间小折线（一般 0.4~0.6）
- POS：副图凯利仓位百分比（合理范围 -1~1，负值代表反向 / 不交易）
- 嵌套调用层数：KELLY ← SMA ← IF ← REF (4 层) — 解析器复合栈深的良好压力测试

---

## 3 · 段 3 — CONNORSRSI 短期反转策略

**含义**：Larry Connors 三因子 RSI（短期 RSI + streak RSI + ROC 百分位）。值 < 10 = 超卖买入，> 90 = 超买卖出。

**公式**：
```
CRSI:CONNORSRSI(CLOSE,3,2,100);
OVERSOLD:CRSI<10;
OVERBOUGHT:CRSI>90;
BUY:CROSS(10,CRSI);
SELL:CROSS(CRSI,90);
```

**预期看到**：
- 副图 CRSI 主线在 [0, 100] 振荡
- OVERSOLD / OVERBOUGHT 阈值线为 0/1 矩形信号
- BUY / SELL 在阈值穿越点跳到 1

---

## 4 · 段 4 — KALMAN 自适应均线交叉系统

**含义**：双卡尔曼快慢线（Q 不同）+ 经典金叉/死叉。Q 越大越跟随价格，越小越平滑。

**公式**：
```
KF_SLOW:KALMAN(CLOSE,0.001,1);
KF_FAST:KALMAN(CLOSE,0.01,1);
GOLDEN:CROSS(KF_FAST,KF_SLOW);
DEATH:CROSS(KF_SLOW,KF_FAST);
```

**预期看到**：
- 主图：CLOSE + KF_SLOW（较平滑）+ KF_FAST（较跟随）— 两条线都贴近 CLOSE 但纹理不同
- 副图：GOLDEN（金叉）/ DEATH（死叉）信号脉冲

---

## 5 · 段 5 — HP_FILTER 趋势/周期分解

**含义**：Hodrick-Prescott 滤波分离价格的「长期趋势」和「短期周期」。lambda=1600 是经济学经典默认（季度数据），用于 K 线相当激进；周期百分位 PCT 标准化到 [0, 100]。

**公式**：
```
TREND:HP_FILTER(CLOSE,1600);
CYCLE:CLOSE-TREND;
HHV20:HHV(CYCLE,20);
LLV20:LLV(CYCLE,20);
PCT:(CYCLE-LLV20)/(HHV20-LLV20)*100;
```

**预期看到**：
- 主图：CLOSE + TREND（很平滑的长期线）
- 副图 1：CYCLE（CLOSE 减去 TREND，正负振荡）
- 副图 2：PCT 百分位 [0, 100]，可作摆动指标

## 已知限制

3. **第 3 段 CONNORSRSI 在前 ~100 根可能为 nil**（streak + 100-period 百分位），属于正常。
